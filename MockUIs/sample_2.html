A page with input

This is a page that handles user input. If you enter an invalid email into the "Email" field and submit it will show you how an error message should look. Otherwise it will show how a success message should look. In addition, it uses custom styling in custom.css on top of bootstrap. You can the forms look different if you want on your UIs.

So, if you submit the information here then it will show you an error message if the email is invalid.
